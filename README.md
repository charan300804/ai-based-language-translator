# 🌐 Translatify - AI-Based Language Translator

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python&style=for-the-badge)](https://www.python.org/)

Translatify is a smart translation tool leveraging free translation APIs with local cached dictionary lookups.

---

## 🚀 Key Features
- **Hybrid System**: Local mock translations and online fallback API integrations.
- **Support Matrix**: Multi-lingual translations (Spanish, French, German, Italian).
